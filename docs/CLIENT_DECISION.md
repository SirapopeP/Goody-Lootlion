# การตัดสินใจฝั่ง Client (Lootlion)

## สรุป

- **เฟส 1 (MVP)**: ใช้ **Angular แบบเว็บ + PWA** (`@angular/pwa`) เป็นหลัก — ติดตั้งบนมือถือผ่าน “Add to Home Screen” ได้ ไม่ต้องแพ็กเกจสโตร์ทันที
- **เฟสถัดไป (ถ้าต้องการ)**: เพิ่ม **Capacitor** (หรือ Ionic บน Angular) เพื่อแพ็กเกจเป็นแอป iOS/Android และต่อ **FCM/APNs** แบบเต็มรูปแบบ

## เหตุผล

- ลดความซับซ้อนและเวลาก่อนมี MVP ใช้งานจริง
- PWA รองรับ offline shell และแจ้งเตือนผ่าน **Web Push** ได้ในระดับหนึ่ง
- Capacitor ใช้ build เดียวกับ Angular — ย้ายได้เมื่อความต้องการชัดเจน
